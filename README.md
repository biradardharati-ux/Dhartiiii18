# 💬 Multi-User Chat Application

## 📌 Project Description
This project is a real-time multi-user chat application that allows multiple users to communicate with each other simultaneously. It uses WebSocket technology for instant messaging and provides an interactive user interface built with HTML, CSS, and JavaScript.

---

## 🚀 Features
- Real-time messaging using WebSockets
- Multiple users can chat simultaneously
- Simple and user-friendly interface
- Dynamic message updates without refreshing the page
- Backend server handling multiple connections

---

## 🛠️ Technologies Used
- Frontend:
  - HTML
  - CSS
  - JavaScript

- Backend:
  - Python

- Protocol:
  - WebSocket
